Place the OECD How's Life? CSV files here.

Download them from the OECD Data Explorer (topic: "Well-being and beyond GDP")
at https://data-explorer.oecd.org/ and save them with these exact filenames:

  current_wellbeing.csv     (Current well-being)            -> used by blocks 1, 2, 5
  vertical_inequality.csv   (Current well-being vertical inequalities) -> block 3
  future_wellbeing.csv      (Future well-being)             -> block 3
  child_wellbeing.csv       (optional, not used in final analysis)
  cwb_by_age.csv            (optional, not used in final analysis)

See the main README for details.
